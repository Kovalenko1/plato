# Запуск приложения

Проект собран на `Vite + React + TypeScript`.

## Команды

**Требования:** Node.js (желательно LTS)

1. Установить зависимости: `npm install`
2. Задать переменную `GEMINI_API_KEY` в `.env.local`
3. Запустить dev-сервер: `npm run dev` (откроется на `http://localhost:3000`)

Сборка и предпросмотр:
- `npm run build`
- `npm run preview`
